# Configuration requise pour DataSuite

## Matériel

Visual Studio Code est un téléchargement léger (< 350 Mo) et occupe moins de 1 Go d’espace disque.  
DataSuite est également léger et devrait fonctionner facilement sur le matériel actuel.

Nous recommandons :

* Processeur de 1,6 GHz ou plus rapide
* 1 Go de RAM

## Plateformes

DataSuite est pris en charge sur les plateformes suivantes :

* Windows 10 et 11 (64 bits)
