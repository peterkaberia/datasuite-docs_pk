# Mode portable

DataSuite prend en charge le [mode portable](https://en.wikipedia.org/wiki/Portable_application).
Ce mode permet à toutes les données créées et gérées par DataSuite d’être stockées à proximité de l’application,
afin qu’elles puissent être déplacées entre différents environnements.

Le mode portable est pris en charge via le téléchargement ZIP pour Windows.
Consultez la [page de téléchargement](/fr/downloads) pour trouver le fichier `.zip` approprié pour votre plateforme.

> [!IMPORTANT]
>
> N’essayez pas de configurer le mode portable sur une installation provenant des installateurs Windows utilisateur
> ou système. Le mode portable est uniquement pris en charge avec l’archive ZIP (.zip) de Windows.
> Notez également que l’archive ZIP Windows ne prend pas en charge la mise à jour automatique.

## Activer le mode portable

### Windows

Après avoir décompressé le téléchargement DataSuite, créez un dossier `data` dans le dossier de DataSuite :

```
|- DataSuite-win32-x64-1.0.0
|  |- Suite.exe
|  |- data
|  |- bin
|  |  |- code
|  |  |- ...
|  |- ...
```

À partir de ce moment, le dossier `data` sera utilisé pour contenir toutes les données DataSuite,
y compris l’état de session, les préférences, etc.

Le dossier `data` peut être déplacé vers d’autres installations de DataSuite.
Ceci est utile lors de la mise à jour de votre version portable de DataSuite,
auquel cas vous pouvez déplacer le dossier `data` vers une version plus récente extraite de DataSuite.

## Mettre à jour DataSuite portable

Sous Windows, vous pouvez mettre à jour DataSuite en copiant le dossier `data`
dans une version plus récente de DataSuite.

## Migrer vers le mode portable

Vous pouvez également migrer une installation existante vers le mode portable.

### Windows

1. Téléchargez la distribution ZIP de DataSuite pour votre plateforme.
2. Créez le dossier `data` comme indiqué ci-dessus.
3. Copiez le répertoire de données utilisateur `DataSuite` dans `data`
   et renommez-le en `user-data`.

   Windows location:
   %APPDATA%\DataSuite

Exemple :

```
|- DataSuite-win32-x64-1.84.2
|  |- DataSuite.exe
|  |- data
|  |  |- user-data
|  |  |  |- ...
|  |- ...
```

## Répertoire TMP

Par défaut, le répertoire TMP reste celui du système même en mode portable,
car aucun état n’y est conservé.

Si vous souhaitez également que votre répertoire TMP soit situé dans votre répertoire portable,
vous pouvez créer un dossier `tmp` vide à l’intérieur du dossier `data`.
Tant qu’un dossier `tmp` existe, il sera utilisé pour les données TMP.
