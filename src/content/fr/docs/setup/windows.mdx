## Installer DataSuite sur Windows [#datasuite-install]

### Utiliser l'installateur Windows

<Steps>
##### Téléchargez l'[installateur DataSuite](/fr/downloads) pour Windows

##### Une fois téléchargé, exécutez l'installateur (DataSuiteUserSetup-{version}.exe).

    Si l'UAC (Contrôle de compte d'utilisateur) vous le demande, cliquez sur **Oui** pour permettre à l'application d'apporter des modifications sur votre appareil.

##### Accepter le contrat de licence

    Parcourez le Contrat de licence de l'utilisateur final (EULA). Sélectionnez **J'accepte le contrat** et cliquez sur **Suivant**.

    ![Accept the License Agreement](/images/install_accept_licence.jpg)

##### Choisir l'emplacement d'installation

    Choisissez le dossier de destination où vous souhaitez installer DataSuite. Par défaut, DataSuite est installé dans `C:\Users\{Username}\AppData\Local\Programs\DataSuite`. Cliquez sur **Suivant** pour continuer.

    ![Choose Installation Location](/images/install_choose_location.jpg)

##### Installer et terminer

    Cliquez sur **Installer** pour lancer le processus. Une fois la barre de progression terminée, cliquez sur **Terminer** pour fermer l'assistant d'installation et lancer DataSuite.

    ![Install and Finish](/images/install_datasuite.jpg)

</Steps>

### Utiliser le fichier ZIP

1. Téléchargez l'[archive ZIP DataSuite](/fr/downloads)

1. Extrayez l'archive ZIP et exécutez DataSuite à partir de là
