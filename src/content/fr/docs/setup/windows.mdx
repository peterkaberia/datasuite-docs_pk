## Installer DataSuite sur Windows [#install-datasuite-on-windows]

### Utiliser l’installateur Windows

<Steps>

##### Télécharger l’[installateur DataSuite](/fr/downloads) pour Windows

##### Une fois téléchargé, exécutez l’installateur (DataSuiteUserSetup-\{version\}.exe).

    Si une invite du Contrôle de compte utilisateur (UAC) apparaît, cliquez sur **Oui** pour autoriser l’application à apporter des modifications à votre appareil.

##### Accepter le contrat de licence

    Lisez le contrat de licence utilisateur final (EULA). Sélectionnez **J’accepte le contrat** puis cliquez sur **Suivant**.

    ![Accepter le contrat de licence](/images/install_accept_licence.jpg)

##### Choisir l’emplacement d’installation

    Choisissez le dossier de destination dans lequel vous souhaitez installer DataSuite. Par défaut, DataSuite est installé dans  
    `C:\Users\{Username}\AppData\Local\Programs\DataSuite`. Cliquez sur **Suivant** pour continuer.

    ![Choisir l’emplacement d’installation](/images/install_choose_location.jpg)

##### Installer et terminer

    Cliquez sur **Installer** pour démarrer le processus. Une fois la barre de progression terminée, cliquez sur **Terminer** pour quitter l’assistant d’installation et lancer DataSuite.

    ![Installer et terminer](/images/install_datasuite.jpg)

</Steps>

### Utiliser le fichier ZIP

1. Téléchargez l’[archive ZIP DataSuite](/fr/downloads)

1. Extrayez l’archive ZIP et lancez DataSuite à partir de ce dossier

## Installation utilisateur versus installation système

DataSuite propose des installations Windows au niveau **utilisateur** et **système**.

| Type d’installation | Description |
|---------------------|-------------|
| [Installation utilisateur](https://github.com/aphrcwaro/datasuite_public/releases/download/1.0.0-beta2/DataSuiteUserSetup-win32-x64-1.0.0-beta2.exe) | Ne nécessite pas de privilèges administrateur, car l’installation se fait dans votre dossier Local AppData (`LOCALAPPDATA`). Comme aucune élévation n’est requise, l’installation utilisateur offre une expérience de mise à jour en arrière-plan plus fluide.<br/>C’est la méthode recommandée pour installer DataSuite sur Windows.<br/> **Remarque :** si DataSuite est exécuté en tant qu’administrateur dans une installation utilisateur, les mises à jour sont désactivées. |
| [Installation système](https://github.com/aphrcwaro/datasuite_public/releases/download/1.0.0-beta2/DataSuiteSetup-x64-1.0.0-beta2.exe) | Nécessite des privilèges administrateur et installe DataSuite dans le dossier système `Program Files`. Le processus de mise à jour intégré nécessite également une élévation, ce qui le rend moins fluide que l’installation utilisateur. En revanche, l’installation système permet à tous les utilisateurs de l’ordinateur d’accéder à DataSuite. |

Consultez la page [Télécharger DataSuite](/fr/downloads) pour obtenir la liste complète des options d’installation disponibles.

## Mise à jour [#update]

Lorsqu’une nouvelle version de DataSuite est publiée avec de nouvelles fonctionnalités et des corrections importantes, vous serez invité à installer la nouvelle version lorsqu’elle devient disponible.

Vous pouvez également vérifier manuellement les mises à jour en sélectionnant **Aide** > **Rechercher des mises à jour** sous Windows.
