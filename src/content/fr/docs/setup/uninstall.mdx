# Désinstaller DataSuite

Les étapes de désinstallation de DataSuite dépendent de votre plateforme et de l'option d'installation que vous avez utilisée. Par exemple, sous Windows, vous pouvez utiliser les installateurs System ou User, ou télécharger un fichier `.zip` et en extraire le contenu n'importe où sur votre machine.

En général, vous devez désinstaller DataSuite comme n'importe quelle autre application de bureau et suivre le flux recommandé par votre plateforme pour supprimer un logiciel. 
Des instructions spécifiques par plateforme sont fournies ci-dessous, ainsi que la manière de [nettoyer complètement](#clean-uninstall) les fichiers de configuration DataSuite restants.

## Windows

### Installateur Windows

Si vous avez installé DataSuite via l'installateur Windows, en version User ou System, utilisez cet installateur pour supprimer DataSuite.

* Menu Démarrer
  * Recherchez **Ajouter ou supprimer des programmes** et trouvez DataSuite dans la liste **Applications** > **Applications et fonctionnalités**.
  * Sélectionnez **Désinstaller** dans le menu des actions à droite (trois points verticaux).
  * Suivez les invites pour désinstaller DataSuite.
* Panneau de configuration
  * Sous **Programmes**, sélectionnez le lien **Désinstaller un programme**.
  * Trouvez l'entrée DataSuite, faites un clic droit et sélectionnez la commande **Désinstaller**.
  * Suivez les instructions pour désinstaller DataSuite.

### Installation via fichier .zip

Si vous avez installé DataSuite sur Windows en téléchargeant et en extrayant l'un des fichiers `.zip` disponibles sur le [site DataSuite], vous pouvez désinstaller le programme en supprimant le dossier dans lequel vous avez extrait le contenu du fichier `.zip`.

## Désinstallation complète

Si vous souhaitez supprimer toutes les données utilisateur après avoir désinstallé DataSuite, vous pouvez supprimer les dossiers de données utilisateur `DataSuite` et `.datasuite`. Cela ramènera votre système à l'état antérieur à l'installation de DataSuite. Cela peut également être utilisé pour réinitialiser toutes les configurations si vous ne souhaitez pas désinstaller DataSuite.

L'emplacement des dossiers varie selon la plateforme :

* **Windows** - Supprimez `%APPDATA%\DataSuite` et `%USERPROFILE%\.datasuite`.
