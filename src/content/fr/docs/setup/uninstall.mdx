# Désinstaller DataSuite

Les étapes pour désinstaller DataSuite dépendent de votre plateforme et de la méthode
d’installation utilisée. Par exemple, sous Windows, vous pouvez utiliser les installateurs
Windows **système** ou **utilisateur**, ou télécharger un fichier `.zip` et extraire son
contenu n’importe où sur votre machine.

En général, vous pouvez désinstaller DataSuite comme n’importe quelle autre application
de bureau en suivant la procédure recommandée par votre plateforme pour désinstaller
un logiciel. Des instructions spécifiques par plateforme sont fournies ci-dessous,
ainsi que la procédure pour [nettoyer complètement](#clean-uninstall) les fichiers de
configuration DataSuite restants.

## Windows

### Installateur Windows

Si vous avez installé DataSuite via l’installateur Windows, soit la version **Utilisateur**
soit la version **Système**, utilisez l’installateur pour supprimer DataSuite.

* Menu Démarrer
  * Recherchez **Ajouter ou supprimer des programmes** et trouvez DataSuite dans la liste
    **Applications** > **Applications et fonctionnalités**.
  * Sélectionnez **Désinstaller** dans le menu d’actions à droite (trois points verticaux).
  * Suivez les instructions pour désinstaller DataSuite.
* Panneau de configuration
  * Dans **Programmes**, sélectionnez le lien **Désinstaller un programme**.
  * Trouvez l’entrée DataSuite, cliquez avec le bouton droit et sélectionnez **Désinstaller**.
  * Suivez les instructions pour désinstaller DataSuite.

### Installation via fichier .zip

Si vous avez installé DataSuite sous Windows en téléchargeant et en extrayant l’un des
fichiers `.zip` disponibles sur le [site DataSuite], vous pouvez désinstaller l’application
en supprimant simplement le dossier dans lequel vous avez extrait le contenu du fichier `.zip`.

## Désinstallation complète [#clean-uninstall]

Si vous souhaitez supprimer toutes les données utilisateur après avoir désinstallé DataSuite,
vous pouvez supprimer les dossiers de données utilisateur `DataSuite` et `.datasuite`.
Cela vous ramènera à l’état initial avant l’installation de DataSuite. Cette méthode peut
également être utilisée pour réinitialiser tous les paramètres si vous ne souhaitez pas
désinstaller DataSuite.

L’emplacement des dossiers dépend de votre plateforme :

* **Windows** - Supprimez `%APPDATA%\DataSuite` et `%USERPROFILE%\.datasuite`.
