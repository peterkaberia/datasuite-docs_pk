# Uninstall DataSuite

The steps for uninstalling DataSuite will depend on your platform and the 
install option that you used. For example on Windows, you can use the System 
or User Windows Installers or download a `.zip` file and extract the contents 
anywhere on your machine.

In general, you would uninstall DataSuite as you would any other desktop application 
and follow your platform's recommended flow for uninstalling software. Specific 
platform guidance is provided below as well as how to [completely clean up](#clean-uninstall) 
any remaining DataSuite configuration files.

## Windows

### Windows Installer

If you installed DataSuite via the Windows Installer, either the User or System version, use 
the installer to remove DataSuite.

* Start menu
  * Search for **Add or Remove Programs** and find DataSuite in the **Apps** > **Apps & features** list.
  * Select **Uninstall** from the actions dropdown on the right side (three vertical dots).
  * Follow the prompts to uninstall DataSuite.
* Control Panel
  * Under **Programs**, select the **Uninstall a program** link.
  * Find the DataSuite entry, right-click, and select the **Uninstall** command.
  * Follow the prompts to uninstall DataSuite.

### .zip file installation

If you installed DataSuite on Windows by downloading and extracting one of the `.zip` files found 
on the [DataSuite website], you can uninstall by deleting the folder where you extracted the `.zip` 
contents.

## Clean uninstall

If you want to remove all user data after uninstalling DataSuite, you can delete the user data 
folders `DataSuite` and `.datasuite`. This will return you to the state before you installed DataSuite. 
This can also be used to reset all settings if you don't want to uninstall DataSuite.

The folder locations will vary depending on your platform:

* **Windows** - Delete `%APPDATA%\DataSuite` and `%USERPROFILE%\.datasuite`.
