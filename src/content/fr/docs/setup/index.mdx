## Configuration du CD2030 DataSuite

`CD2030 DataSuite` fonctionne actuellement sur les systèmes d'exploitation Windows. C'est une plateforme légère qui vous permet de l'installer ou d'utiliser une version portable et de démarrer vos opérations de données en quelques minutes.

Pour commencer, suivez les instructions de configuration étape par étape ci-dessous :

<Steps>
### Vérifiez les [Exigences système](/fr/docs/supporting/requirements)
- Pour vous assurer que votre matériel et la configuration de votre ordinateur sont compatibles

### Téléchargez et installez `CD2030 DataSuite` pour votre plateforme
- Sélectionnez l'option d'[installateur/configuration](/fr/downloads/#installer-options) adaptée à votre PC.
- [Installez](/fr/docs/setup/windows) l'application `CD2030 DataSuite`

### Accédez à [Commencer avec DataSuite](/fr/docs/getting-started)
- Pour découvrir l'interface utilisateur (UI) et les principales fonctionnalités de `CD2030 DataSuite`
</Steps>
