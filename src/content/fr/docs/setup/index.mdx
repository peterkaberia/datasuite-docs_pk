# Configuration de DataSuite

DataSuite fonctionne principalement sur les systèmes d’exploitation Windows, avec un support pour d’autres plateformes.  
La mise en route est rapide et simple. C’est une plateforme légère qui vous permet de l’installer et de commencer vos opérations de données en quelques minutes.

Avant l’installation, vous pouvez consulter les [Configuration requise](/fr/docs/supporting/requirements) afin de vérifier que votre matériel et la configuration de votre ordinateur sont compatibles.

## Fréquence des mises à jour [#update-cadence]

Lorsqu’une nouvelle version de DataSuite est publiée avec de nouvelles fonctionnalités et des corrections importantes, 
vous serez invité à installer la nouvelle version lorsqu’elle devient disponible.

Vous pouvez également vérifier manuellement les mises à jour en sélectionnant **Aide** > **Rechercher des mises à jour** sous Windows.
## Mode portable

DataSuite prend en charge l’installation en [mode portable](https://en.wikipedia.org/wiki/Portable_application).  
Ce mode permet à toutes les données créées et maintenues par DataSuite d’être stockées à proximité de l’application, 
afin qu’elles puissent être déplacées facilement entre différents environnements, par exemple sur une clé USB.  
Consultez la documentation [Mode portable de DataSuite](/fr/docs/supporting/portable) pour plus de détails.

### Comment effectuer une désinstallation « propre » de DataSuite ?

Si vous souhaitez supprimer toutes les données utilisateur après avoir [désinstallé](/fr/docs/setup/uninstall) DataSuite, vous pouvez supprimer les dossiers de données utilisateur `DataSuite` et `.datasuite`.  
Cela vous ramène à l’état initial avant l’installation de DataSuite.  
Cela peut également être utilisé pour réinitialiser tous les paramètres si vous ne souhaitez pas désinstaller DataSuite.

Les emplacements des dossiers varient selon votre plateforme :

* **Windows** - Supprimez `%APPDATA%\DataSuite` et `%USERPROFILE%\.datasuite`.
