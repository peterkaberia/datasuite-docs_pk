 **Q. How do I know which version of `CD2030 Datasuite` I'm running?**
   - On the Menu bar, right-click **Help** and  select  **About**.​​​​​​​ ![Version check](/images/datasuite_version_check.jpg)

**Q. How can I do a 'clean' uninstall of DataSuite?**
- If you want to remove all user data after [uninstalling](/en/docs/setup/uninstall) DataSuite, you can delete 
the user data folders `DataSuite` and `.datasuite`. 
- This returns you to the state before you installed DataSuite. This approach can also be used to reset all settings if you don't want to uninstall DataSuite.
- The folder locations vary depending on your platform:

      **Windows** - Delete `%APPDATA%\DataSuite` and `%USERPROFILE%\.datasuite`.
