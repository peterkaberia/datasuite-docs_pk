# Framework Overview

This page summarizes the CD2030 DataSuite analytical framework.
It is intended to help teams move from raw facility and survey data to defensible national and sub-national coverage findings.

## End-to-end workflow

The framework follows a practical sequence:

1. Load and configure facility and survey data.
2. Calibrate key national benchmark rates.
3. Assess data quality before estimating coverage.
4. Adjust incomplete, unstable, or unreliable data.
5. Select the best-performing denominator.
6. Interpret national, equity, and sub-national results.
7. Use findings for planning, review, and action.

See the [workflow page](/en/docs/framework/workflow) for the process view.
