# Requirements for DataSuite

## Hardware

CD2030 DataSuite is a small download (< 350 MB) and has a disk footprint of < 1 GB. 
DataSuite is lightweight and should easily run on today's hardware.

We recommend:

* 1.6 GHz or faster processor
* 1 GB of RAM

## Platforms

DataSuite is supported on the following platforms:

* Windows 10 and 11 (64-bit)
