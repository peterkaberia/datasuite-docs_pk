# Setting up CD2030 DataSuite

**CD2030 DataSuite** currently runs on Windows operating systems, with support for other platforms. 
Getting up and running is quick and easy. It is a lightweight platform, allowing you to 
install it and begin your data operations in a matter of minutes.

Before installing, you can review the [System Requirements](/en/docs/supporting/requirements)
to ensure your hardware and computer configuration are supported.

## Update cadence [#update-cadence]

When DataSuite releases a new version with new features and important bug fixes you are prompted to 
install the new release when it becomes available.

You can also manually check for updates by running **Help** > **Check for Updates** on Windows.

## Portable mode

DataSuite supports [Portable mode](https://en.wikipedia.org/wiki/Portable_application) installation. 
This mode enables all data created and maintained by DataSuite to live near itself, so it can be moved 
around across environments, for example, on a USB drive. See the 
[DataSuite Portable Mode](/en/docs/supporting/portable) documentation for details.

### How can I do a 'clean' uninstall of DataSuite?

If you want to remove all user data after [uninstalling](/en/docs/setup/uninstall) DataSuite, you can delete 
the user data folders `DataSuite` and `.datasuite`. This returns you to the state before you installed DataSuite. 
This can also be used to reset all settings if you don't want to uninstall DataSuite.

The folder locations vary depending on your platform:

* **Windows** - Delete `%APPDATA%\DataSuite` and `%USERPROFILE%\.datasuite`.
