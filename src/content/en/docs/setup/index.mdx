## Setting up CD2030 DataSuite

`CD2030 DataSuite` currently runs on Windows operating systems. It is a lightweight platform, allowing you to 
install it or use a portable version and begin your data operations in a matter of minutes.

To get started, follow the following step-by-step set up instructions:


<Steps>
### Review the [System Requirements](/en/docs/supporting/requirements) 
- To ensure your hardware and computer configuration are supported

### Download and install `CD2030 DataSuite` for your platform
- Select the correct [installer/set up](/en/downloads/#installer-options) option for your PC.
- [Install](/en/docs/setup/windows) the `CD2030 Datasuite` App

### Go to [Get started with the DataSuite](/en/docs/getting-started) 
- To have a glimpse of the user interface (UI) and key features of the `CD2030 DataSuite`

</Steps>
