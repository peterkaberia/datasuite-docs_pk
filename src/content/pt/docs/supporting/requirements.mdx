# Requisitos para o DataSuite

## Hardware

O Visual Studio Code é um download pequeno (< 350 MB) e ocupa menos de 1 GB de espaço em disco.  
O DataSuite é leve e deve funcionar facilmente no hardware atual.

Recomendamos:

* Processador de 1,6 GHz ou mais rápido
* 1 GB de RAM

## Plataformas

O DataSuite é suportado nas seguintes plataformas:

* Windows 10 e 11 (64 bits)
