## Definições dos indicadores

O Data Suite padroniza as definições de indicadores em todos os sistemas de informação de gestão de saúde (HMIS / SIGS) para permitir uma análise consistente entre países. Esta secção especifica exatamente quais as variáveis do HMIS que mapeiam para os indicadores de saúde reprodutiva, materna, neonatal, infantil e do adolescente (SRMNIA) — prevenindo erros de interpretação e garantindo que os dados das suas unidades de saúde alimentam os numeradores e denominadores corretos. As definições padronizadas também permitem a comparação entre países e ao longo do tempo, tornando mensurável o progresso global em direção à cobertura universal de saúde.

Três categorias de dados organizam os requisitos: dados administrativos (inventário de unidades de saúde, força de trabalho), dados populacionais (estimativas demográficas, taxas de sobrevivência) e dados de serviço (eventos notificados por unidades, estratificados por indicador e nível administrativo). Uma vez que as estruturas do HMIS dos países variam muito, esta referência permite-lhe extrair ou mapear sistematicamente os seus dados para os nomes de variáveis padronizados que o Data Suite espera. A adesão rigorosa a estas definições é fundamental para uma análise fiável.

## Definições dos indicadores

Para a padronização da análise entre países, esta secção descreve os indicadores/variáveis extraídos dos HMIS dos países, que são necessários para a análise do Datasuite.  
Os dados estão agrupados nas seguintes categorias principais:

- **Dados Administrativos (Admin Data)**
- **Dados Populacionais (Population Data)**
- **Completude de Notificação (Reporting Completeness)**
- **Dados de Serviço 1, 2 e 3 (Service Data)**
- **Verificações de Dados (Data Checks)**

---

### Dados Administrativos (Admin Data)

As variáveis seguintes descrevem a infraestrutura do sistema de saúde e os profissionais.

| Nome da Variável | Descrição |
|--------------|------------|
| `Num_order` | Nº |
| `District_name` | Distrito |
| `First_admin_level` | Região |
| `Country` | O nome do seu país |
| `Number_hospitals` | Número total de hospitais (com base na definição do país) |
| `Number_hcenters` | Número total de centros de saúde (não hospitais; com base na definição do país) |
| `Total_number_health_facilities` | Número total de unidades de saúde (todos os tipos/categorias) |
| `Number_pfacilities_profit` | Número total de unidades de saúde que são privadas com fins lucrativos |
| `Number_pfacilities_nonprofit` | Número total de unidades de saúde que são privadas sem fins lucrativos |
| `Total_physicians` | Número total de médicos / oficiais médicos |
| `Total_nurses_midwives` | Número total de enfermeiros / parteiras |
| `Total_clinical_officers` | Número total de oficiais clínicos / assistentes médicos |
| `Total_nonclinique_physicians` | Número total de médicos não clínicos |
| `Number_hospital_beds` | Número de camas hospitalares |

### Verificações de Dados (Data Checks)

Variáveis de diagnóstico automatizadas utilizadas para validação de completude e consistência em todas as folhas.

| Nome da Variável | Descrição |
|--------------|------------|
| `District` | Nome do Distrito |
| `# of occurrences in 'Admin data'` | Contagem de verificação da presença do distrito na folha de Dados Administrativos (Admin Data) |
| `Is the district missing in 'Population_data'?` | Sinalizador de estado verificando se o distrito está em falta nos Dados Populacionais |
| `# of occurrences in 'Population_data'?` | Contagem de verificação da presença do distrito nos Dados Populacionais |
| `Is the district missing in 'Reporting_completeness'?` | Sinalizador de estado verificando se o distrito está em falta na Completude de Notificação |
| `# of occurrences in 'Reporting completeness'` | Contagem de verificação da presença do distrito na Completude de Notificação |
| `Is the district missing in 'Service_data_1'?` | Sinalizador de estado verificando se o distrito está em falta nos Dados de Serviço 1 |
| `# of occurrences in 'Service_data_1'` | Contagem de verificação da presença do distrito nos Dados de Serviço 1 |
| `Is the district missing in 'Service_data_2'?` | Sinalizador de estado verificando se o distrito está em falta nos Dados de Serviço 2 |
| `# of occurrences in 'Service_data_2'` | Contagem de verificação da presença do distrito nos Dados de Serviço 2 |
