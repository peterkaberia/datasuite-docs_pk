## Abordagem de modelagem bayesiana

### Contexto
Informações confiáveis sobre os indicadores de cobertura em saúde são essenciais para monitorar o progresso, orientar programas e subsidiar decisões de política. No entanto, depender apenas de dados de inquérito e de rotina é frequentemente desafiador. Os dados de rotina das unidades de saúde podem sofrer subnotificação ou qualidade de dados inconsistente, enquanto os dados de inquérito podem ser incertos ou não recentes.

Para enfrentar esses desafios, modelos estatísticos podem ser usados para produzir estimativas e previsões ao longo do tempo, combinando múltiplas fontes de dados e considerando suas limitações. Isso resulta em estimativas de cobertura para cada ano (entre pontos de dados e dados passados) com uma indicação de quão certos/incertos estamos sobre cada ano. Como resultado, os modelos permitem avaliações de cobertura em saúde mais completas, precisas e oportunas, especialmente em contextos com dados limitados.

---

### Produção de estimativas e previsões de curto prazo
Desenvolvemos um modelo estatístico para estimar a cobertura em saúde no nível nacional e para regiões subnacionais até o ano mais recente com dados, e para fazer previsões de curto prazo para os anos seguintes.

O modelo analisa os dados disponíveis e identifica diferentes formas pelas quais a cobertura pode ter mudado ao longo do tempo. Cada um desses caminhos possíveis é chamado de **trajetória** (ver Figura 1).

Para cada ano, relatamos uma melhor estimativa (o valor médio entre todas as trajetórias) e um intervalo que mostra a incerteza (excluindo as possibilidades mais extremas na borda).

Informado por análises globais, o modelo captura que, ao longo do tempo, a cobertura geralmente sobe de níveis baixos para níveis altos, seguindo uma tendência em forma de S, com períodos de progresso mais rápido ou mais lento dependendo do que os dados indicam. Ele usa dados de inquérito e de rotina para escolher trajetórias que correspondam ao observado. O modelo também se baseia em tendências observadas em outras áreas com níveis de cobertura semelhantes.

### Como os dados de inquérito são usados?
Os dados de inquérito ajudam a entender a cobertura, mas vêm com alguma incerteza. Isso acontece porque as inquéritos incluem apenas uma amostra da população, não todos, e podem ocorrer erros durante a coleta de dados.

Para tirar o melhor proveito dos dados de inquérito, o modelo tenta encontrar o equilíbrio certo — suavizando altos e baixos aleatórios que provavelmente não são reais, enquanto ainda captura padrões importantes nos dados. Para isso, o modelo analisa quão incerto cada ponto de dados é:
* **Alta incerteza:** Se um ponto de dados for muito incerto, ele terá menos impacto na cobertura estimada (o modelo dá menos peso a ele e tende a suavizá-lo).
* **Alta confiabilidade:** Se um ponto de dados for mais confiável, as estimativas ficarão mais próximas desse valor.

Para regiões subnacionais, o modelo usa dados de inquérito da região específica quando disponíveis.

### Como os dados de rotina são usados?
As estimativas de cobertura a partir de dados de rotina podem ser viesadas, por isso o modelo os trata de forma diferente dos dados de inquérito. Para as inquéritos, o modelo usa os níveis reais de cobertura. Por exemplo, se um inquérito indica cobertura de 60% para um ano específico, o modelo usa esse valor diretamente.

Em contraste, os dados de rotina podem ser viesados devido a problemas como notificação incompleta e inconsistências na coleta de dados. Por isso, o modelo não confia nos níveis de cobertura observados nos dados de rotina. Em vez disso, o modelo se concentra na **tendência ao longo do tempo** — especificamente, como a cobertura muda de ano para ano.

* **Exemplo:** Se os dados de rotina mostram a cobertura aumentando de 50% em 2020 para 53% em 2021, o modelo interpreta isso como um aumento de 3 pontos percentuais. Se os dados de rotina mostram mudanças mais rápidas ou mais lentas do que o esperado, o modelo ajusta as estimativas — mas apenas na medida em que os dados são confiáveis.

<Callout type="info">
**Consideração da qualidade dos dados**

O modelo leva em conta a qualidade dos dados de rotina ajustando a incerteza das estimativas de tendência. Observações com qualidade de dados inferior são tratadas como mais incertas e recebem menos peso na determinação das taxas reais de mudança. Especificamente, a incerteza na taxa de mudança estimada aumenta à medida que a pontuação média de qualidade dos dados do Countdown diminui. Consequentemente, dados de menor qualidade exercem menos influência na taxa de mudança modelada.
</Callout>

### Exemplos ilustrativos

As estimativas são produzidas para:
* A proporção de mulheres que recebem o mínimo recomendado de pelo menos quatro consultas de cuidados pré-natais durante a gravidez (**ANC4**).
* A proporção de nascimentos vivos que ocorrem em uma unidade de saúde (**Partos institucionais**).
* A proporção de crianças que recebem as três doses recomendadas da vacina pentavalente (**Cobertura vacinal**).
