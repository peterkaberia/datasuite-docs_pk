# Leveraging AI for Analysis and Interpretation

Artificial intelligence (AI) can enhance the efficiency, depth, and accessibility of data analysis and interpretation.

## AI as a Support Tool

AI can help analysts to:
- Explore datasets quickly  
- Identify patterns and trends  
- Generate summaries and visualizations  
- Structure written outputs  

However, AI must be used with professional judgment and cannot replace expert interpretation.

## AI Roles and Limitations

| AI can amplify | AI must not replace |
|---------------|-------------------|
| Identifying patterns and trends | Deciding what questions to ask |
| Detecting data quality issues | Assessing data reliability |
| Generating summaries | Interpreting findings in context |
| Producing draft outputs | Making final decisions |

## What Makes an Effective Prompt

- Be clear on purpose  
- Define the audience  
- Specify geography and scope  
- Provide interpretation guidance  
- Specify output format  
- Set guardrails  

## Example Prompt

> Using only the data provided, summarize key trends, identify patterns and outliers, and highlight gender and equity differences. Clearly distinguish between analysis and interpretation and flag limitations.
