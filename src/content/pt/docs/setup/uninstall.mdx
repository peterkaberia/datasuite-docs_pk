# Desinstalar o DataSuite

As etapas para desinstalar o DataSuite dependem da sua plataforma e da opção de
instalação utilizada. Por exemplo, no Windows, você pode usar os instaladores
Windows **System** ou **User**, ou baixar um arquivo `.zip` e extrair seu conteúdo
em qualquer local da sua máquina.

Em geral, você deve desinstalar o DataSuite como qualquer outro aplicativo de
desktop, seguindo o fluxo recomendado pela sua plataforma para remover softwares.
Orientações específicas por plataforma são fornecidas abaixo, bem como instruções
sobre como [limpar completamente](#clean-uninstall) quaisquer arquivos de configuração
restantes do DataSuite.

## Windows

### Instalador do Windows

Se você instalou o DataSuite usando o instalador do Windows, seja a versão **User**
ou **System**, utilize o próprio instalador para remover o DataSuite.

* Menu Iniciar
  * Pesquise **Adicionar ou remover programas** e encontre o DataSuite na lista
    **Apps** > **Apps e recursos**.
  * Selecione **Desinstalar** no menu de ações à direita (três pontos verticais).
  * Siga as instruções para desinstalar o DataSuite.
* Painel de Controle
  * Em **Programas**, selecione o link **Desinstalar um programa**.
  * Encontre a entrada do DataSuite, clique com o botão direito e selecione **Desinstalar**.
  * Siga as instruções para desinstalar o DataSuite.

### Instalação via arquivo .zip

Se você instalou o DataSuite no Windows baixando e extraindo um dos arquivos `.zip`
disponíveis no [site do DataSuite], você pode desinstalar o programa simplesmente
excluindo a pasta onde extraiu o conteúdo do arquivo `.zip`.

## Desinstalação completa [#clean-uninstall]

Se você quiser remover todos os dados do usuário após desinstalar o DataSuite,
pode excluir as pastas de dados do usuário `DataSuite` e `.datasuite`.
Isso retornará o sistema ao estado anterior à instalação do DataSuite.
Isso também pode ser usado para redefinir todas as configurações caso você
não queira desinstalar o DataSuite.

A localização das pastas varia de acordo com a plataforma:

* **Windows** - Exclua `%APPDATA%\DataSuite` e `%USERPROFILE%\.datasuite`.
