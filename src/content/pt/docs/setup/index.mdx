# Configurando o DataSuite

O DataSuite funciona principalmente em sistemas operacionais Windows, com suporte para outras plataformas.  
A configuração é rápida e fácil. É uma plataforma leve que permite instalar e iniciar suas operações de dados em poucos minutos.

Antes da instalação, você pode revisar os [Requisitos do Sistema](/pt/docs/supporting/requirements)
para garantir que seu hardware e configuração do computador são compatíveis.

## Frequência de atualizações [#update-cadence]

Quando o DataSuite lança uma nova versão com novos recursos e correções importantes de erros, 
você será solicitado a instalar a nova versão quando ela estiver disponível.

Você também pode verificar manualmente se há atualizações acessando **Ajuda** > **Verificar atualizações** no Windows.

## Modo portátil

O DataSuite oferece suporte à instalação em [modo portátil](https://en.wikipedia.org/wiki/Portable_application).  
Este modo permite que todos os dados criados e mantidos pelo DataSuite fiquem armazenados próximos ao próprio aplicativo, 
permitindo que sejam movidos entre ambientes, por exemplo, em uma unidade USB.  
Consulte a documentação [Modo portátil do DataSuite](/pt/docs/supporting/portable) para mais detalhes.

### Como saber qual versão estou usando?

No Windows, selecione **Ajuda** > **Sobre**.

### Como fazer uma desinstalação “limpa” do DataSuite?

Se você quiser remover todos os dados do usuário após [desinstalar](/pt/docs/setup/uninstall) o DataSuite, 
você pode excluir as pastas de dados do usuário `DataSuite` e `.datasuite`.  
Isso retorna o sistema ao estado anterior à instalação do DataSuite.  
Isso também pode ser usado para redefinir todas as configurações caso você não queira desinstalar o DataSuite.

Os locais das pastas variam dependendo da sua plataforma:

* **Windows** - Exclua `%APPDATA%\DataSuite` e `%USERPROFILE%\.datasuite`.
